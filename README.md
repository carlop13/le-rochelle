# 🏡 Le Rochelle

**Le Rochelle** es un sitio web inmobiliario desarrollado con **CodeIgniter y MySQL**, diseñado para que agencias y particulares puedan **publicar, administrar y consultar propiedades en línea**.  
La plataforma incluye funciones de **geolocalización**, **gestión de usuarios**, **galerías de imágenes** y generación de **contratos digitales** para agilizar las operaciones de compra, venta o renta.

---

## 🚀 Características principales
- 🔑 Registro e inicio de sesión de clientes y agentes inmobiliarios.  
- 🏠 Publicación y gestión de propiedades (casas, departamentos, terrenos, locales).  
- 📸 Galerías de imágenes para cada propiedad.  
- 🌍 Geolocalización de propiedades mediante Google Maps API.  
- 🔎 Búsqueda avanzada con filtros (precio, ubicación, tipo de propiedad, características).  
- 📄 Generación de contratos en PDF para rentas y ventas.  
- 📱 Diseño responsive compatible con dispositivos móviles.  

---

## 🛠️ Tecnologías utilizadas
- **Framework backend**: [CodeIgniter 3](https://codeigniter.com/)  
- **Base de datos**: MySQL  
- **Frontend**: HTML5, CSS3, JavaScript (Bootstrap)  
- **Geolocalización**: Google Maps API  
- **Servidor**: Apache / Nginx con PHP
- **PHP**: 7

---

## 🖥️ Uso del sistema

- Los agentes inmobiliarios publican propiedades con fotos, ubicación y detalles.
- Los clientes consultan propiedades usando filtros avanzados y mapa interactivo.
- El sistema permite gestionar citas y enviar mensajes.
- Al concretar una operación, se genera un contrato en PDF.
